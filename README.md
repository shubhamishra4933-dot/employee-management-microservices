# Employee Management Microservices

## Overview
Production-style Employee Management System built using Java, Spring Boot, JWT Authentication, PostgreSQL, Eureka Server, API Gateway, and Microservices Architecture.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- JWT
- PostgreSQL
- Eureka Server
- Spring Cloud Gateway
- OpenFeign
- Maven

## Microservices
- Auth Service
- Employee Service
- Department Service
- API Gateway
- Service Registry

## Features
- Employee CRUD Operations
- Department CRUD Operations
- JWT Authentication
- Role Based Authorization
- Service Discovery
- API Gateway Routing
